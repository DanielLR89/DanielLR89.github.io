# Proyecto ALLSHOES

## Descripción

Este repositorio contiene los archivos HTML y CSS del proyecto ALLSHOES, un sitio web de venta de zapatos.

## Páginas Disponibles

- [Inicio](index.html)
- [Sobre Nosotros](html/sobre-nosotros.html)
- [Cursos](html/cursos.html)
- [Tienda](html/tienda.html)
- [Contacto](html/contacto.html)

## Cómo Utilizar

1. Clona este repositorio.
2. Abre los archivos HTML en tu navegador web para visualizar las páginas.
3. Modifica los archivos HTML y CSS según sea necesario para adaptar el sitio web a tus necesidades.
